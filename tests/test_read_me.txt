This is how to run our tests

Javascript Test README
1. Install Node.js
2. in CS561/pages/ folder, run 'npm init' to install modules (mocha and chai)
3. in CS561/test/js_test/ folder, open index.html in browser

Intro: How to add new test case in js_test.
Assuming you have a hello_world.js need to test
1. write your own test file test_hello_world.js
2. add hello_world.js to index.html in js_test folder.
    <script src="../../pages/js/hello_world.js"></script>
3. add test_hello_world.js to index.html in js_test folder
    <script src="test_hello_world.js"></script>
